# biblioteca-elotech-app
Projeto Angular para a Bilioteca

# Recursos
- CRUD para usuários

# Observações
- Para rodar o projeto  primeiramente 
 - executar no terminal yarn install
 - npm start 
- Disponivel em http://localhost:4200/
- A rota para acesso a API de bilioteca fica em 
    - src\environments\environment.ts
    - apiUrlBase:'http://localhost:8080/api/'
